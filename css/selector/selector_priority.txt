css 의 선택자 우선순위
1. 동급의 선택자를 썼을 땐 마지막에 작성한 선택자가 적용된다.
2. 같은 것을 가리키더라도 * > tag > class > id 순으로 우선순위가 결정된다. 선택자를 부르는 범위가 좁을 수록 우선순위가 높다.
tag나 class는 '누구 옆에 사는 누구'라고 한다면 id는 특정 이름을 부르는 거와 같다. 이름이 구체적인 면에서 더 큼.
3. id 보다 우선순위가 높은 것은 inline style 이다. html 의 태그 안에 직접 style 을 넣은 것을 가리킨다. 
4. 최고로 우선순위가 높은 것은 !important 이다.
